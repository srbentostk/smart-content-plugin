# Checklist Pré-Escrita

Verificar ANTES de começar a escrever qualquer roteiro.

## Tema
- [ ] O tema está definido claramente?
- [ ] O tema é universal o suficiente? (conflitos que todo mundo vive)
- [ ] O tema carrega carga emocional PRÉVIA? (público já sente algo sobre isso)
- [ ] Se é nicho, tem potencial de atravessar a bolha?

## Ideia central
- [ ] Qual é a crença que o vídeo vai instalar/reforçar?
- [ ] Consigo dizer em 1 frase? (Se não, não está claro)
- [ ] Essa crença é relevante para o público?

## Público
- [ ] Sei quem vai assistir?
- [ ] Conheço a linguagem da bolha?
- [ ] Sei que conflitos/polêmicas existem nesse universo?
- [ ] Sei que referências culturais o público reconhece?

## Conflito
- [ ] Identifiquei o conflito que vai gerar emoção?
- [ ] Esse conflito vai acontecer na CABEÇA do público ou só no roteiro?
- [ ] O conflito é universal o suficiente para ativar carga emocional prévia?

## Formato
- [ ] Escolhi estrutura de roteiro adequada?
- [ ] O formato suporta o tipo de conteúdo? (visual, câmera, tela verde)

## Restrições
- [ ] Não estou inventando nada (personagens, histórias, dados)
- [ ] Se usar pessoa real, é conhecida e verificável
- [ ] Tenho fonte para dados/histórias que vou usar
