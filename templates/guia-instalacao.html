Introdução
O que é a Sofia?

A Sofia é uma assistente inteligente que vive dentro do Claude (uma inteligência artificial). Ela faz coisas que você levaria horas fazendo manualmente — em segundos.

🔍

Pesquisa de vídeos

Você manda o link de um vídeo viral e ela descobre por que aquele vídeo funcionou — frase por frase, com explicações sobre o que prende a atenção das pessoas.

✍️

Escrita de roteiros

Ela te ajuda a escrever o texto do seu vídeo, fazendo perguntas para entender o que você quer dizer, sugerindo várias opções de abertura, e escrevendo versões completas.

Em detalhe
O que a Sofia consegue extrair de um vídeo?

Quando você manda o link de um vídeo, a Sofia vai até a internet, coleta diversas informações daquele vídeo, e traz tudo organizado para você. Veja o que ela busca:

📝

Texto falado no vídeo

Ela pega tudo que a pessoa falou no vídeo e transforma em texto escrito. É como ter a transcrição completa do que foi dito.

💬

Comentários mais curtidos

Ela pega os comentários que mais receberam curtidas. Isso mostra o que o público mais sentiu ao assistir o vídeo.

📊

Números do vídeo

Quantas pessoas assistiram, quantas curtiram, quantas comentaram. Esses números mostram o alcance do vídeo.

🖼️

Capa do vídeo

A imagem de capa (thumbnail) que aparece antes de clicar no vídeo. Útil para entender como o vídeo atrai o clique.

🎯

Primeira frase (hook)

A frase de abertura é separada e analisada. É ela que faz a pessoa parar de rolar o feed e prestar atenção.

🧠

Análise frase a frase

Cada frase do vídeo é marcada com o princípio que ela usa para prender a atenção — curiosidade, conflito, familiaridade, etc.

De onde vêm essas informações? A Sofia usa uma ferramenta chamada yt-dlp que vai até o site do vídeo (YouTube, Instagram, TikTok) e coleta essas informações automaticamente. É como se ela "lesse" a página do vídeo e trouxesse tudo para você. Essa ferramenta é 100% gratuita e funciona com milhares de sites.

Por trás
Como a Sofia busca essas informações?

A Sofia usa duas ferramentas para coletar dados dos vídeos. Ela sempre tenta a ferramenta gratuita primeiro.

Ferramenta principal 100% grátis

yt-dlp

Grátis — sem conta, sem limite de uso

É um programa que roda no seu computador e coleta informações de vídeos. Funciona com milhares de sites.

No YouTube, ele pega TUDO que a Sofia precisa:

  • O texto falado no vídeo (transcrição completa)
  • Os comentários mais curtidos (com a quantidade de curtidas de cada um)
  • Os números do vídeo (visualizações, curtidas, data)
  • A capa (imagem de thumbnail)

No Instagram e TikTok, ele pega dados parciais:

  • ✅ Curtidas do vídeo, descrição, capa, áudio
  • ❌ Não traz as curtidas de cada comentário (dado essencial para entender o público)
  • ❌ Traz poucos comentários e não traz as visualizações

Por isso, para Instagram e TikTok, a Sofia complementa com o Apify (veja abaixo).

Necessário para Instagram Tem créditos grátis

Apify

~R$0,15 por vídeo — $5/mês grátis (~150 análises)

É um serviço online que coleta dados completos de redes sociais. Para Instagram (a plataforma mais usada pelos criadores), ele é importante porque traz o que o yt-dlp não consegue:

  • Os comentários mais curtidos COM a quantidade de curtidas — isso mostra o que o público mais sentiu ao ver o vídeo
  • O número de visualizações do Reel
  • Dados extras como compartilhamentos e salvamentos

Preço: Ao criar a conta, você ganha $5 por mês em créditos grátis. Cada vídeo custa ~$0.03 (3 centavos de dólar). Com os créditos grátis, dá para analisar uns 150 vídeos por mês sem pagar.

Para YouTube: Não precisa do Apify — o yt-dlp já faz tudo.

Criar conta: apify.com (recomendado se usa Instagram)

🎯 Resumo por plataforma:

YouTube → yt-dlp faz tudo (100% grátis)

Instagram → yt-dlp pega o áudio e a capa + Apify pega os comentários com curtidas e as visualizações

TikTok → igual ao Instagram

A Sofia nunca gasta dinheiro sem perguntar antes. Se o Apify não estiver configurado, ela avisa e mostra o que conseguiu pegar só com o yt-dlp.

Escolha como instalar

Tem 3 opções. Escolha a que for mais fácil para você:

✨

Opção A: Pelo Claude Code (mais simples)

Se já tem Claude Code instalado

  1. Abra o Terminal e digite claude
  2. Dentro do Claude, digite:
    /plugin install smart-content-plugin
  3. Pronto! A Sofia está instalada.
🖱️

Opção B: Dois cliques (instala tudo)

Se não tem nada instalado ainda

  1. Acesse a página do projeto
  2. Baixe Instalar Sofia.app (Mac) ou Instalar Sofia.vbs (Windows)
  3. Dê dois cliques no arquivo baixado
  4. Siga as instruções na tela
⌨️

Opção C: Comando no Terminal (instala tudo)

Se prefere usar o Terminal diretamente

  1. Abra o Terminal (Mac) ou PowerShell (Windows)
  2. Cole o comando abaixo e aperte Enter
curl -sSL https://raw.githubusercontent.com/srbentostk/smart-content-plugin/main/scripts/install.sh | bashclique para copiar

Não tem Claude Code? É a versão do Claude que roda no Terminal. Instale com:

claude.ai/download → (Mac/Windows/Linux)

Ou as opções B e C acima já instalam o Claude Code automaticamente.

O que as opções B e C instalam automaticamente:

  • ✅ Node.js — programa necessário para rodar o Claude Code
  • ✅ Claude Code — a versão do Claude que roda no Terminal
  • ✅ yt-dlp — ferramenta grátis que coleta dados de vídeos
  • ✅ Arquivos da Sofia — baixados para o seu computador

Uso
O que você pode fazer com a Sofia

Depois de instalar, você usa a Sofia digitando comandos simples. São como "botões" que você escreve. Aqui está tudo que ela faz:

/sofia-analisar Analisar um vídeo. Você cola o link e ela busca o texto falado, os comentários mais curtidos, os números e faz uma análise completa do que faz aquele vídeo funcionar. Grátis
/sofia-roteiro Escrever um roteiro. Ela faz perguntas sobre seu tema, seu público e o que as pessoas estão comentando. Depois sugere aberturas e escreve 2-3 versões do roteiro completo. Grátis
/sofia-critica Criticar um roteiro. Você cola um roteiro (seu ou de outra pessoa) e ela avalia com honestidade: está bom? Está chato? O que está faltando? Grátis
/sofia-melhorar Melhorar um roteiro. Depois de uma crítica, ela reescreve o roteiro corrigindo os problemas que encontrou. Grátis
/sofia-dashboard Ver painel visual. Gera uma página bonita com a análise completa de um vídeo — perfeita para estudar. Grátis
/sofia-exportar Salvar e aprender. Exporta tudo que a Sofia aprendeu em arquivos organizados e faz ela ficar mais inteligente a cada análise. Grátis

Sobre custos: Todas as funções são grátis. Para analisar vídeos do YouTube, é 100% grátis. Para vídeos do Instagram e TikTok, a Sofia usa o Apify para pegar os comentários com curtidas (~R$0,15 por vídeo) — mas o Apify dá $5 grátis por mês, o que dá para umas 150 análises sem pagar nada.

Dúvidas
Perguntas frequentes
Preciso pagar alguma coisa? ›

A Sofia em si é grátis. Você precisa de uma conta no Claude (grátis ou Pro a $20/mês). O yt-dlp é 100% grátis. Para analisar vídeos do YouTube, não gasta nada. Para Instagram/TikTok, o Apify é recomendado (~R$0,15 por vídeo), mas ele dá $5 grátis por mês — o que cobre umas 150 análises.

Funciona no celular? ›

Não. A Sofia precisa do Claude Code, que roda no Terminal do computador (Mac ou Windows). Não funciona no celular por enquanto.

Funciona no navegador (claude.ai)? ›

Não. A versão do Claude no navegador não permite instalar plugins como a Sofia. Você precisa do Claude Code (que roda no Terminal). O comando de instalação já instala o Claude Code para você.

O que é o Terminal? ›

É um programa que já vem no seu computador. Parece uma tela escura onde você digita comandos. No Mac, chama "Terminal". No Windows, chama "PowerShell". Você só precisa abrir uma vez para colar o comando de instalação — depois disso, a Sofia cuida de tudo.

Com quais sites a Sofia funciona? ›

A ferramenta que a Sofia usa (yt-dlp) funciona com milhares de sites. Os principais são: YouTube, Instagram, TikTok, Twitter/X, Reddit, Facebook, Twitch e Vimeo. Se o site tiver vídeo público, provavelmente funciona.

A instalação deu erro, o que faço? ›

Feche o Terminal e abra novamente. Cole o comando de instalação de novo. Se continuar dando erro, tire um print da mensagem de erro e envie para seu instrutor — ele vai te ajudar.

O que acontece se o yt-dlp não conseguir pegar um vídeo? ›

A Sofia avisa o que conseguiu e o que não conseguiu pegar. Se o Apify estiver configurado, ela usa automaticamente para completar (os créditos grátis cobrem ~150 análises/mês). Se não, ela pergunta se você quer configurar ou se prefere colar o texto do vídeo manualmente.

Preciso do Apify para usar a Sofia? ›

Para YouTube: Não precisa. O yt-dlp faz tudo de graça. Para Instagram e TikTok: É muito recomendado. Sem o Apify, a Sofia não consegue pegar os comentários mais curtidos (com a quantidade de curtidas de cada um) — e isso é essencial para entender o que o público sentiu. O Apify tem $5 grátis por mês, o que dá para analisar ~150 vídeos sem pagar.

Referência
Glossário — Palavras usadas neste guia
Terminal
Programa que já vem no computador. Parece uma tela escura onde você digita comandos. No Mac: "Terminal". No Windows: "PowerShell".
yt-dlp
Ferramenta gratuita que coleta dados de vídeos. No YouTube pega tudo (texto, comentários com curtidas, números). No Instagram/TikTok pega dados parciais (áudio, capa, curtidas do vídeo).
Apify
Serviço online que complementa o yt-dlp no Instagram e TikTok — traz os comentários com curtidas e as visualizações. Tem $5/mês grátis (~150 análises).
Claude Code
Versão do Claude que roda no Terminal. É mais poderosa que a versão do navegador e permite instalar plugins como a Sofia.
Hook
A primeira frase do vídeo. É ela que faz a pessoa parar de rolar o feed e prestar atenção.
Roteiro
O texto completo do que vai ser falado no vídeo, do começo ao fim.
Princípios
As regras de psicologia que fazem um vídeo prender a atenção — curiosidade, conflito, familiaridade, contraste, etc.
Bolha
O grupo de pessoas que segue o mesmo tipo de conteúdo. Exemplo: a bolha de empreendedorismo, a bolha de fitness, etc.
Transcrição
O texto de tudo que foi falado no vídeo, transformado de áudio para texto escrito.
Thumbnail
A imagem de capa do vídeo — aquela que aparece antes de clicar para assistir.
Dashboard
Uma página visual que mostra a análise do vídeo toda organizada e bonita.
API Key
Um código secreto (como uma senha) que permite um programa acessar um serviço. Só é necessário para o Apify, e é opcional.
