# Estruturas de Roteiro

> **Este diretório será preenchido com o conteúdo do Trello (board: Formatos Criativos de Conteúdo - lista "Estruturas de Roteiro").**

## Como preencher

1. Exportar ou copiar cada card da lista "Estruturas de Roteiro" do Trello
2. Cada estrutura deve ter seu próprio arquivo .md
3. Formato sugerido:

```markdown
# [Nome da Estrutura]

## Quando usar
- [Contexto ideal para esta estrutura]

## Esqueleto
1. [Passo 1 — função no roteiro]
2. [Passo 2 — função no roteiro]
3. [...]

## Princípios que ativa naturalmente
- [Lista de princípios que esta estrutura favorece]

## Exemplo de referência
- [Link ou descrição de vídeo que usa esta estrutura]

## Dicas de aplicação
- [Como maximizar eficácia com esta estrutura]
```

## Estruturas conhecidas (do Google Doc)

Baseado nos exemplos analisados, estas estruturas já foram identificadas:

1. **Tela Dividida** — Pessoa falando com imagens/vídeo de apoio ao lado
2. **Storytelling Visual** — Imagens ilustram a narrativa em tempo real
3. **Tela Verde** — Background muda para contextualizar a história
4. **Câmera Direta** — Pessoa fala direto para câmera, sem edição elaborada
5. **Narração de Cena** — Narra o que está acontecendo (ex: MasterChef)
